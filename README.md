# fold-and-bond
Interactive computational biology card game simulating protein folding, secondary structure formation, amino acid interactions, and structural motifs using Python and Pygame.

# Fold & Bond
### Interactive Protein Folding Card Game

An interactive computational biology and scientific visualization project built using Python and Pygame.

Fold & Bond is an educational protein folding card game that simulates:

- protein secondary structure formation
- amino acid chemistry
- hydrophobic collapse
- salt bridges
- disulfide bonding
- folding stability
- structural motifs
- tertiary interactions

The project combines computational biology, biochemical heuristics, procedural visualization, and game design into a playable educational experience.

---

# Features

## Procedural Protein Structure Visualization

Each amino acid card contains procedurally generated structural artwork rendered in real time using Pygame, including:

- α-helices
- β-sheets
- random coils
- glycine flexibility states
- proline helix-breaking kinks
- cysteine disulfide bridges

---

## Biochemical Interaction Engine

The folding engine implements biologically inspired scoring systems including:

- Chou-Fasman secondary structure propensities
- Kyte-Doolittle hydrophobicity
- hydrogen bonding
- hydrophobic packing
- salt bridges
- aromatic stacking
- structural propagation
- motif recognition

---

## Gameplay Systems

- Mission-based folding objectives
- Stability and entropy mechanics
- Folding failure detection
- Tertiary structure bonuses
- Real-time interaction networks
- Dynamic protein fold scoring

# Scientific Inspiration

This project is inspired by foundational concepts in structural biology and protein chemistry including:

- Chou-Fasman secondary structure prediction
- Kyte-Doolittle hydrophobicity scales
- protein motif recognition
- supersecondary structure formation
- tertiary stabilization interactions

This is an educational simulation inspired by computational structural biology concepts and is not intended to represent physically accurate molecular dynamics or protein folding prediction.
